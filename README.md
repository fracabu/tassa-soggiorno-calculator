<a name="english"></a>
## :gb: English

### Overview

A web application designed to automatically calculate the Rome tourist tax for 2025, supporting exemptions and data export.

### Features

- **2025 Rates** - Updated Rome tax rates
- **Exemptions** - Support for tax exemptions
- **Data Export** - Export calculations
- **Multiple Properties** - Various property types
- **Guest Categories** - Adults, children, exempt guests

### Quick Start

```bash
git clone https://github.com/fracabu/tassa-soggiorno-calculator.git
cd tassa-soggiorno-calculator
# Open index.html in browser
```

### Panoramica
